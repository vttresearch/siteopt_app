# Adding data to Siteopt tool

This directory contains the configuration of input files for Spine Toolbox.

Install the Siteopt tool first using the instructions provided. If **current_input** subfolder does not exist in the Siteopt folder, create it. Then clone the input data from this repository to the current_input subfolder. Do not create additional folder levels. Files should reside directly in the current_input subfolder.
